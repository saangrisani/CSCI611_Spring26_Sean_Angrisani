# Assignment 2

## Overview
This project contains two parts:
1. Image filtering implementation
2. CNN construction and evaluation

## Files
- `image_filter.ipynb` - Part 1 implementation
- `build_cnn.ipynb` - Part 2 implementation
- `report.pdf` - Final report
- `figures/` - Graphs and visualizations used in the report

## How to Run
1. Open the notebooks in Jupyter Notebook or JupyterLab
2. Run all cells from top to bottom
3. Make sure required Python packages are installed:
   - numpy
   - matplotlib
   - torch
   - torchvision
   - scikit-learn
   - time

## Outputs
The notebooks include:
- execution traces
- graphs
- tables
- accuracy results
- training time
- CNN parameter counts
